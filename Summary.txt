Summary

* Added Lab 6: a B-Tree implementation in C++ (`StorageEngine/lab6/main.cpp`) with a menu-driven interactive driver.
* Implemented a `BTree` class parameterized by the minimum degree `t`.
* Each non-root node maintains between `t - 1` and `2t - 1` keys.
* Implemented insertion using the standard top-down preemptive split strategy.
* `splitChild(parent, idx, child)` promotes the median key to the parent and creates a new right sibling containing the keys and children after the median.
* `insertNonFull(node, key)` inserts keys into non-full nodes and proactively splits full children before descending.
* Root expansion occurs only when the root itself becomes full.
* Implemented key lookup using recursive B-Tree search.
* Implemented inorder traversal to display keys in sorted order.
* Added interactive menu options for Insert, Search, Display, and Quit.
